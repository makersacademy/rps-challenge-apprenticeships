# ROCK PAPER SCISSORS WEB APP

## Installing locally

```ruby
$ git clone https://github.com/khushnoodasif/rps-challenge-apprenticeships.git
$ cd rps-challenge-apprenticeships
$ bundle install
$ rackup
```
## Run the app

```ruby 
$ ruby app.rb
```
or

```ruby 
$ rackup config.ru
```
